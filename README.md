---
## 📌 **Description**
Script Batch pour **masquer des fichiers** sous Windows 10.
- **Personnalisable** : Choisissez le dossier et les extensions à masquer.

**Dernière mise à jour** : 18 décembre 2024

---
## 📁 **Fichiers**
- `Masque.bat` : Script principal

---
## ⚙️ **Personnalisation**
Modifiez le script pour :
- Définir le **dossier d’exécution**
- Ajouter les **extensions de fichiers** à masquer

---
## 🚀 **Utilisation**
1. Ouvrez `Masque.bat` avec un éditeur de texte
2. Modifiez les paramètres
3. Exécutez le script en tant qu’administrateur

---
## ⚠️ **Avertissement**
> ⚠️ Les fichiers masqués restent accessibles via l’explorateur de fichiers (option "Afficher les fichiers cachés").
